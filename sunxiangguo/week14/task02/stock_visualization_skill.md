# 股票可视化分析 Skill

## 功能描述

对股票数据进行可视化分析，包括：
1. 绘制股票的周波动和日波动曲线（在同一图中）
2. 基于波动幅度分析，给出买入卖出的最佳时间建议

## 使用场景

- 分析单只股票的价格走势
- 对比不同时间粒度的波动情况
- 获取基于技术分析的交易建议

## 快速开始

### 1. 安装依赖

```bash
pip install requests pandas numpy matplotlib
```

### 2. 运行脚本

```bash
# 基本用法（默认分析贵州茅台2024年数据）
python scripts/stock_visualization.py

# 指定股票代码
python scripts/stock_visualization.py 600519.SH

# 指定股票代码和日期范围
python scripts/stock_visualization.py 300750.SZ 2024-06-01 2024-12-31
```

### 3. Python代码调用

```python
from scripts.stock_visualization import analyze_stock

# 分析贵州茅台
recommendation = analyze_stock("600519.SH", "2024-01-01", "2024-12-31")

# 分析宁德时代
recommendation = analyze_stock("300750.SZ", "2024-06-01", "2024-12-31")
```

## 输入参数

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| stock_code | string | 股票代码 | "600519.SH" 或 "300750.SZ" |
| start_date | string | 开始日期 | "2024-01-01" |
| end_date | string | 结束日期 | "2024-12-31" |

## 输出内容

### 1. 可视化图表

生成 `stock_XXXXXX_analysis.png` 文件，包含4个子图：
- **主图**：日线收盘价 + 周线收盘价 + MA20/MA60均线
- **副图1**：日波动率（5日滚动标准差）
- **副图2**：成交量（红涨绿跌）
- **副图3**：日收益率百分比

### 2. 分析报告

控制台输出详细的交易建议，包括：
- 📊 基本统计（当前价格、最高/最低价、整体趋势）
- 📈 波动性分析（日/周波动率、波动特征分类）
- 💡 关键技术位（阻力位、支撑位、当前位置判断）
- 🎯 交易建议（买入时机、卖出时机）
- ⚠️ 风险控制（止损位、仓位管理、分批操作）

## 核心功能说明

### 数据获取

使用 `akshare` 库获取A股历史数据，支持前复权处理。

### 数据处理

- **日线数据**：直接使用原始交易数据
- **周线数据**：按周聚合（开盘价取首值、最高/最低取极值、收盘价取末值、成交量求和）
- **技术指标**：计算MA5/MA20/MA60均线、日/周波动率、收益率

### 买卖时机算法

**买入信号识别：**
1. 价格回调至支撑位附近且波动率降低
2. 日线突破MA20均线且成交量放大
3. 周线显示连续3周以上企稳迹象
4. 波动率低于历史平均值

**卖出信号识别：**
1. 价格接近阻力位且出现滞涨信号
2. 日线跌破MA20均线且成交量萎缩
3. 单日波动率超过平均值的2倍
4. 达到预期收益目标（15-20%）

### 支撑阻力位计算

- **阻力位** = 期间最高价 × 0.98
- **支撑位** = 期间最低价 × 1.02

## 注意事项

1. **数据源**：使用akshare获取A股数据，确保网络连接正常
2. **中文字体**：首次运行可能需要安装中文字体（SimHei或Microsoft YaHei）
3. **免责声明**：分析结果仅供参考，不构成投资建议
4. **参数调整**：可根据需要修改均线周期和波动率计算窗口
5. **代理设置**：如访问数据源需要代理，请提前配置

## 常见问题

**Q: 如何分析港股或美股？**
A: 当前版本仅支持A股（.SH/.SZ），如需分析其他市场需修改数据获取部分。

**Q: 图表中文显示乱码怎么办？**
A: 安装中文字体或在代码中修改 `plt.rcParams['font.sans-serif']` 为系统可用字体。

**Q: 如何自定义分析周期？**
A: 通过命令行参数或函数调用时传入不同的 start_date 和 end_date。

**Q: 波动率高低如何判断？**
A: 
- 低波动：< 1.5%
- 中等波动：1.5% - 3%
- 高波动：> 3%
